# Weather_DiscordBot

## 概要
天気予報を表示するDiscord Botです。<br/>
[OpenWeather](https://openweathermap.org/)を参照しています。

## 使い方
```
git clone https://github.com/keito0tada/Weather_DiscordBot.git
cd Weather_DiscordBot
pip install -r requirements.txt
python weather.py
```
